#Buffer source

##Description
Create a playable audio source from a buffer of audio data.

##Inputs
###buffer
An audio buffer to create a playable source from.

###play
Start or stop playback.

###loop
Looping playback.

###loop-start
Start of loop (in seconds).

###loop-end
End of loop (in seconds).

##Outputs
###source
A playable audio source
